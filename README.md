## Run Locally
Clone the project

Go to the project directory

```bash
  cd hackathon-ideas
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

Link to access - please run only on the port 3000
```bash
  http://localhost:3000
```
